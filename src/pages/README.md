# Pages — INCyTEA-APP

| Ruta | Componente | Función |
|---|---|---|
| `/` | `Landing.jsx` | Página de bienvenida con hero, tarjetas informativas y botón para ingresar al sistema. |
| `/login` | `Login.jsx` | Selección de rol y formulario de acceso (simulado, sin validación real). Guarda el rol en `localStorage`. |
| `/dashboard` | `Dashboard.jsx` | Panel principal con tarjetas de estadísticas y tabla de actividad reciente. |
| `/vehicular` | `Vehicular.jsx` | Gestión de flota: fichas de permisos de salida e inventario de vehículos. |
| `/caseta` | `Caseta.jsx` | Control de acceso: registro de entradas/salidas, búsqueda de folios, historial y ticket. |
| `/admin` | `Admin.jsx` | Administración de personal: CRUD de cuentas con roles. |
| `/dirysec` | `DirySec.jsx` | Vale de salida para Dirección/Secretaría con campos de formulario, firmas en canvas e impresión. |
